# Temperature and Humidity Display

This Arduino sketch reads temperature and humidity data from a DHT11 sensor and displays it on a 16x2 LCD.

## Installation:

1. Download the library file DHT_sensor_library-1.4.6.zip.
2. Extract the contents of the zip file.
3. Copy the extracted folder to the following directory: `Documents\Arduino\libraries\paste here`.

## Usage:

1. Connect a DHT11 sensor to pin 8 of your Arduino board.
2. Connect a 16x2 LCD to the specified pins in the sketch.
3. Upload the sketch to your Arduino board.

## Dependencies:

This sketch relies on the DHT sensor library. Please make sure to install the library as per the installation instructions provided above.
